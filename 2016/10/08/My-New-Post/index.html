hexo的简单搭建基础 

我也是首次搭建hexo静态网页，目前只有浅显的研究，后期将更加深入研究。参考搭建简介http://www.jianshu.com/p/465830080ea9，http://www.tuicool.com/articles/ueI7naV

hexo简介

Hexo 是一个简单地、轻量地、基于Node的一个静态博客框架，可以方便的生成静态网页托管在github和Heroku上，引用Hexo作者 @tommy351 的话：

快速、简单且功能强大的 Node.js 博客框架。A fast, simple & powerful blog framework, powered by Node.js.

hexo搭建要素

搭建hexo需要3个前提软件。
1、node.js这个可以在node官网下载https://nodejs.org/en/，主要是用来生成静态页面的
2、git，可以把本地的hexo内容提交到github上去的作用
3、nodepad++，主要是用来编辑配置文件的

以上软件下载下来之后只要一路next就可以了，选择默认设置

hexo搭建步骤

1、hexo安装
新建一个文件夹存放hexo全部配置文件，然后右键鼠标选择 git bash here，然后全局安装hexo，敲入命令
npm install -g hexo就可以了，输入命令hexo init ["博客名称"]会生成一个标题为“博客名称”的博客。
2、键入命令 hexo generate会生成一个静态页面
3、键入 hexo server 启动本地服务可以进行文章预览调试，服务地址为：http://localhost:4000

以上简单的静态页面就搭建完成了。
